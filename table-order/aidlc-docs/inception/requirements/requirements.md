## 기능 요구사항 (Functional Requirements)

### FR-01: 테이블 태블릿 자동 로그인 및 세션 관리
- 관리자가 태블릿 초기 설정 시 매장 식별자, 테이블 번호, 테이블 비밀번호 입력
- 로그인 정보 로컬 저장 후 자동 로그인
- 태블릿 비밀번호는 기기 보안 잠금용 (무단 설정 변경 방지)

### FR-02: 메뉴 조회 및 탐색
- 메뉴 화면이 기본 화면으로 항상 표시
- 카테고리별 메뉴 분류 및 표시
- 메뉴 상세 정보: 메뉴명, 가격, 설명, 이미지
- 카테고리 간 빠른 이동
- 카드 형태 레이아웃, 터치 친화적 (최소 44x44px)

### FR-03: 장바구니 관리
- 메뉴 추가/삭제, 수량 조절
- 총 금액 실시간 계산
- 장바구니 비우기
- 클라이언트 측 로컬 저장 (새로고침 시에도 유지)
- 서버 전송은 주문 확정 시에만

### FR-04: 주문 생성
- 주문 내역 최종 확인 후 주문 확정
- 주문 성공 시: 주문 번호 표시 (5초) → 장바구니 비우기 → 메뉴 화면 자동 리다이렉트
- 주문 실패 시: 에러 메시지 표시, 장바구니 유지
- 주문 정보: 매장 ID, 테이블 ID, 메뉴 목록, 총 금액, 세션 ID

### FR-05: 주문 내역 조회
- 현재 테이블 세션 주문만 표시
- 주문 시간 순 정렬
- 주문별: 주문 번호, 시각, 메뉴/수량, 금액, 상태 (대기중/준비중/완료)
- 매장 이용 완료 처리된 주문 제외

### FR-06: 매장 관리자 인증
- 매장 식별자 + 사용자명 + 비밀번호 로그인
- 16시간 세션 유지 (JWT 토큰)
- 브라우저 새로고침 시 세션 유지
- 16시간 후 자동 로그아웃
- 비밀번호 bcrypt 해싱
- 로그인 시도 제한

### FR-07: 실시간 주문 모니터링
- SSE 기반 실시간 주문 목록 업데이트
- 그리드/대시보드 레이아웃 (테이블별 카드)
- 각 테이블 카드: 총 주문액, 최신 주문 미리보기
- 주문 카드 클릭 시 전체 메뉴 목록 상세 보기
- 주문 상태 변경 (대기중/준비중/완료)
- 신규 주문 시각적 강조 (색상, 애니메이션)
- 2초 이내 주문 표시

### FR-08: 테이블 관리
- 테이블 초기 설정 (번호, 비밀번호, 16시간 세션)
- 주문 삭제 (확인 팝업 → 즉시 삭제 → 총 주문액 재계산)
- 테이블 세션 종료 (이용 완료): 주문 내역 과거 이력 이동, 현재 목록/총액 리셋
- 과거 주문 내역 조회 (시간 역순, 날짜 필터링)

### FR-09: 메뉴 관리
- 메뉴 CRUD (등록/조회/수정/삭제)
- 메뉴 정보: 메뉴명, 가격, 설명, 카테고리, 이미지 URL
- 메뉴 노출 순서 조정
- 필수 필드 검증, 가격 범위 검증

## 비기능 요구사항 (Non-Functional Requirements)

### NFR-01: 성능
- SSE 기반 실시간 주문 알림 2초 이내 전달
- 메뉴 조회 응답 시간 1초 이내
- 동시 접속: 단일 매장 기준 테이블 20개 + 관리자 1~2명

### NFR-02: 보안
- JWT 토큰 기반 인증
- 비밀번호 bcrypt 해싱
- 로그인 시도 제한 (brute-force 방지)
- HTTPS 통신 (TLS 1.2+)
- 입력값 검증 및 SQL Injection 방지

### NFR-03: 가용성
- 매장 영업 시간 동안 안정적 서비스 제공
- 서버 장애 시 클라이언트 측 장바구니 데이터 유지

### NFR-04: 사용성
- 터치 친화적 UI (최소 44x44px 터치 영역)
- 직관적 네비게이션
- 명확한 시각적 피드백 (성공/실패)

### NFR-05: 유지보수성
- 모노레포 구조로 통합 관리
- 명확한 모듈 분리 (프론트엔드/백엔드)
- API 문서화

## 제외 범위 (Out of Scope)

다음 기능은 구현하지 않음:
- 결제 처리 (PG사 연동, 영수증, 환불, 포인트/쿠폰)
- 복잡한 인증 (OAuth, SNS 로그인, 2FA)
- 이미지 리사이징/최적화, CMS, 광고
- 알림 시스템 (푸시, SMS, 이메일, 소리/진동)
- 주방 기능 (주방 전달, 재고 관리)
- 고급 기능 (분석 대시보드, 매출 리포트, 직원 관리, 예약, 리뷰, 다국어)
- 외부 연동 (배달 플랫폼, POS, 소셜 미디어, 지도, 번역)

## 데이터 모델 개요

### 핵심 엔티티
- **Store**: 매장 정보
- **Table**: 테이블 정보 (매장 소속)
- **TableSession**: 테이블 세션 (시작~이용완료)
- **Category**: 메뉴 카테고리
- **Menu**: 메뉴 항목
- **Order**: 주문 (세션 소속)
- **OrderItem**: 주문 항목 (메뉴별 수량/단가)
- **OrderHistory**: 과거 주문 이력
- **Admin**: 관리자 계정

| 용어 | 정의 |
|------|------|
